# my-gulp-skel

My skelton for gulp, webpack, CoffeeScript, TypeScript, Stylus, etc.

## Directory Configuration 

* src/
  + js/
    - *.js, *.ts, *.coffee
  + css/
    - *.css, *.styl
  + data/
    - *.html, *.png, etc.
* gulp/
  + config.coffee
    - user settings
* tmp/
  + (compiled) temporary files
* build/
  + production files

## Usage

```
$ gulp
$ gulp watch
$ gulp clean
$ gulp clobber
```
