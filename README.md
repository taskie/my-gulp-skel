# my-gulp-skel

My skelton for gulp, webpack, CoffeeScript, TypeScript, Babel, Jade, Stylus, etc.

## Directory Configuration 

* src/
  + js/
    - *.js, *.ts, *.coffee
  + css/
    - *.css, *.styl
  + html/
	- *.html, *.jade
  + data/
    - *.png, *.jpg, *.ttf, etc.
* gulp/
  + config.coffee
    - user settings
* tmp/
  + (compiled) temporary files
* build/
  + production files

## Usage

```
$ npm update -D
```

```
$ gulp
$ gulp watch
$ gulp clean
$ gulp clobber
```
